# Python Zari App

Μια απλή και διασκεδαστική desktop εφαρμογή σε Python που προσομοιώνει το ρίξιμο ενός ζαριού. Την έφτιαξα κυρίως για να εξασκηθώ στα (GUI) και να δω πώς δουλεύουν τα απλά animations!

## Πώς δουλεύει
Αντί να πετάει απλά έναν αριθμό, όταν πατάς το κουμπί το ζάρι αλλάζει γρήγορα σύμβολα για λίγο πριν σταματήσει στο τελικό αποτέλεσμα. 

* Δεν χρησιμοποίησα εξωτερικές εικόνες jpg για να μην είναι βαρύ. Αντί γι' αυτό, έπαιξα με τα Unicode σύμβολα κειμένου (⚀ ⚁ ⚂ ⚃ ⚄ ⚅) και τους έδωσα τεράστιο font size!
* Όσο το ζάρι γυρίζει, το κουμπί κλειδώνει για να μην μπορείς να το σπαμάρεις.
* Στο τέλος, ο αριθμός που έτυχε γίνεται πράσινος για να ξεχωρίζει.

## Τι χρησιμοποίησα
Όλα είναι γραμμένα με τις built-in βιβλιοθήκες της Python:
* `tkinter` (για το παράθυρο και τα κουμπιά)
* `random` (για να διαλέγει στην τύχη τα πρόσωπα του ζαριού)
* `time` (το `time.sleep()` κάνει τη δουλειά για το εφέ της καθυστέρησης στο animation)
